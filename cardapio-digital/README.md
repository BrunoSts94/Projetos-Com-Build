## 📋 Previsão  do Tempo
Projeto pessoal para estudo e melhor compreensão sobre React
## 🧐 Sobre
Esse é um projeto de uma página apresentando um cardapio digital de uma lanchonete ficticia, o objetivo aqui foi estudar fundamentos do React como uso de useStates, reutilização de componentes e boas práticas do codigo limpo
## 🤖🛠️ Tecnologias e ferramentas utilizadas
- HTML5
- CSS3
- Javascript
- React
- Node js
